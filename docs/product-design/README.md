# Aurelis Product Design

Versión 1.0

Estado: En desarrollo

---

Foundation nos permitió descubrir qué es Aurelis.

Product Design tiene un propósito diferente.

Convertir esa visión en experiencias reales.

Aquí ya no discutiremos la identidad del producto.

La identidad quedó definida durante Foundation.

A partir de este momento diseñaremos cómo cada músico vivirá esa identidad.

Cada recorrido.

Cada pantalla.

Cada interacción.

Cada mensaje.

Cada decisión de diseño deberá responder una pregunta muy sencilla.

¿Cómo ayudamos a que el músico dedique menos tiempo a organizarse y más tiempo a hacer música?

Product Design representa el puente entre la visión del producto y la ingeniería.

Cuando esta etapa concluya, el desarrollo dejará de imaginar el producto.

Comenzará únicamente a construirlo.

---

"No diseñamos interfaces.

Diseñamos momentos que ayudan a hacer música."
