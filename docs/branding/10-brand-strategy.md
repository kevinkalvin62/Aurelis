# Introducción

Una gran marca no se construye únicamente con un buen producto.

Se construye ocupando un lugar claro en la mente de las personas.

Este documento define cómo Aurelis desea ser percibida por el mundo.

No describe campañas.

No describe publicidad.

Describe la estrategia que dará coherencia a toda nuestra comunicación durante los próximos años.
